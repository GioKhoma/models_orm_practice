# ­ЪД▒ DJANGO MODELS & ORM PRACTICE

## ­ЪЊЮ рЃАрЃљрЃ«рЃћрЃџрЃЏрЃФрЃдрЃЋрЃљрЃюрЃћрЃџрЃЮ: Django рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

рЃљрЃЏ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ Django-рЃА **рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЮрЃЉрЃћрЃЉрЃўрЃА** рЃАрЃгрЃљрЃЋрЃџрЃљ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ. рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃљрЃдрЃгрЃћрЃарЃўрЃџрЃўрЃљ рЃАрЃљрЃЏрЃў рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃбрЃўрЃърЃўрЃА рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЮрЃЉрЃљ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃљрЃЊ.

---
## ­ЪјЊ рЃАрЃљрЃЋрЃљрЃарЃ»рЃўрЃерЃЮ: рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃАрЃарЃБрЃџрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ

### рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў:
РюЁ **рЃЏрЃўрЃюрЃўрЃЏрЃБрЃЏ 6 рЃЏрЃЮрЃЊрЃћрЃџрЃў**  
РюЁ **рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃАрЃљрЃЏрЃўрЃЋрЃћ рЃбрЃўрЃърЃўрЃА рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЮрЃЉрЃљ**  
РюЁ **рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў, рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ**


## рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃћрЃЉрЃўрЃЊрЃљрЃю рЃљрЃЏрЃЮрЃўрЃарЃЕрЃўрЃћрЃЌ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃў рЃЌрЃЦрЃЋрЃћрЃюрЃЌрЃЋрЃўрЃА рЃАрЃљрЃАрЃарЃБрЃЋрЃћрЃџрЃў рЃЊрЃљ рЃљрЃљрЃњрЃћрЃЌ рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў


## ­ЪћЌ 1. OneToOne рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЮрЃЉрЃљ

**рЃарЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ?** рЃарЃЮрЃфрЃљ рЃћрЃарЃЌрЃў рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃў рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃљрЃЊ рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌ рЃАрЃ«рЃЋрЃљ рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃА.

### ­ЪЊї рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃћрЃЉрЃў:
- рЃцрЃўрЃџрЃЏрЃў Рєћ IMDB рЃарЃћрЃўрЃбрЃўрЃюрЃњрЃў (Movie Рєћ IMDBRating)
- рЃЌрЃљрЃюрЃљрЃЏрЃерЃарЃЮрЃЏрЃћрЃџрЃў Рєћ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃЏрЃљрЃњрЃўрЃЊрЃљ (Employee Рєћ Desk)
- рЃЏрЃљрЃюрЃЦрЃљрЃюрЃљ Рєћ рЃАрЃљрЃюрЃЮрЃЏрЃарЃћ рЃюрЃўрЃерЃљрЃюрЃў (Car Рєћ LicensePlate)
- рЃгрЃўрЃњрЃюрЃў Рєћ ISBN рЃЎрЃЮрЃЊрЃў (Book Рєћ ISBNCode)

**­ЪњА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃў:** рЃћрЃарЃЌрЃў Рєњ рЃћрЃарЃЌрЃў (1:1)

---

## ­ЪћЌ 2. ForeignKey рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЮрЃЉрЃљ (One-to-Many)

**рЃарЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ?** рЃарЃЮрЃфрЃљ рЃЉрЃћрЃЋрЃарЃў рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃў рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ рЃћрЃарЃЌ рЃЏрЃерЃЮрЃЉрЃћрЃџ рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃА.

### ­ЪЊї рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃћрЃЉрЃў:
- рЃљрЃЋрЃбрЃЮрЃарЃў Рєњ рЃАрЃбрЃљрЃбрЃўрЃћрЃЉрЃў (Author Рєњ Articles)
- рЃЎрЃЮрЃЏрЃърЃљрЃюрЃўрЃљ Рєњ рЃЌрЃљрЃюрЃљрЃЏрЃерЃарЃЮрЃЏрЃџрЃћрЃЉрЃў (Company Рєњ Employees)
- рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ Рєњ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў (Category Рєњ Products)
- рЃцрЃЮрЃџрЃЊрЃћрЃарЃў Рєњ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў (Folder Рєњ Files)
- рЃљрЃџрЃЉрЃЮрЃЏрЃў Рєњ рЃАрЃўрЃЏрЃдрЃћрЃарЃћрЃЉрЃў (Album Рєњ Songs)

**­ЪњА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃў:** рЃћрЃарЃЌрЃў Рєњ рЃЉрЃћрЃЋрЃарЃў (1:N)

---

## ­ЪћЌ 3. ManyToMany рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЮрЃЉрЃљ

**рЃарЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ?** рЃарЃЮрЃфрЃљ рЃЉрЃћрЃЋрЃарЃў рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃў рЃЮрЃарЃўрЃЋрЃћ рЃЏрЃ«рЃарЃўрЃЊрЃљрЃю рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃА.

### ­ЪЊї рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃћрЃЉрЃў:
- рЃцрЃўрЃџрЃЏрЃћрЃЉрЃў Рєћ рЃЏрЃАрЃљрЃ«рЃўрЃЮрЃЉрЃћрЃЉрЃў (Movies Рєћ Actors)
- рЃ»рЃњрЃБрЃцрЃў Рєћ рЃЏрЃЮрЃюрЃљрЃгрЃўрЃџрЃћрЃћрЃЉрЃў (Team Рєћ Members)
- рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃў Рєћ рЃЎрЃБрЃарЃАрЃћрЃЉрЃў (Students Рєћ Courses)

**­ЪњА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃў:** рЃЉрЃћрЃЋрЃарЃў Рєћ рЃЉрЃћрЃЋрЃарЃў (N:M)
